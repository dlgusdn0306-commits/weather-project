# 🌦 나만의 날씨 예보 웹사이트

OpenWeatherMap API를 활용한 날씨 조회 웹사이트입니다.  
도시명을 입력하면 현재 날씨 정보를 불러오고, 도시 이름에 따라 전체 배경 사진이 바뀌며  
날씨 상태(맑음/구름/비/눈 등)에 따라 오른쪽 아래에 아이콘이 표시됩니다.

---

## 🚀 배포 주소 (Vercel)

👉 https://weather-project-lemon-seven.vercel.app

---

## 🛠 사용 기술

- **HTML**: 페이지 구조, 검색창, 결과 표시 영역 구성
- **CSS**: 가운데 카드 UI, 반투명 글래스 효과, 도시별 배경 이미지 처리
- **JavaScript**
  - OpenWeatherMap API 연동
  - 도시 검색 기능
  - 배경/아이콘 자동 업데이트 기능

- **Git / GitHub**: 버전 관리 및 소스 업로드
- **Vercel**: 자동 배포 및 호스팅

---

## 📌 주요 기능

### ✔ 도시 검색
사용자가 입력한 도시의 현재 날씨를 API를 통해 받아와 출력합니다.

### ✔ 도시별 배경 이미지 변경
예: Seoul → 서울 야경, London → 런던, New York → 뉴욕

### ✔ 날씨 상태 아이콘
맑음(clear), 구름(clouds), 비(rain), 눈(snow) 상태에 따라 오른쪽 아래에 아이콘 표시
